\subsection{Application Usage}

Students are expected to implement missing components of a 3D Halfedge Mesh editor, focusing on geometry traversal and subdivision algorithms.  
The application provides an interactive GUI for loading, editing, and subdividing polygonal meshes.

\begin{itemize}
    \item \textbf{File Loading}: Import OBJ meshes via GUI dialog
    \item \textbf{Camera Navigation}: Orbit, pan, and zoom around the scene
    \item \textbf{Selection}: Click to select mesh objects
    \item \textbf{Operations}: Apply triangulation, linear subdivision, Loop subdivision, or Catmull--Clark subdivision
    \item \textbf{Visualization}: Toggle wireframe, vertex normals, face normals
    \item \textbf{Scene Management}: Insert/delete objects, reset camera
\end{itemize}
